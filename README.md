# organizador-de-arquivos-python
Um software em Python simples com uma baita interface gráfica (zueira) para organizar arquivos com base na sua extensão, com tratamento de duplicatas e erros.

Este projeto é um organizador de arquivos inteligente desenvolvido em Python. Ele foi criado como um auxílio no meu desenvolvimento como programador, onde testei meus conhecimentos em um projeto prático, com interface gráfica e lógica de manipulação de sistemas de arquivos.

Com esse projeto, quis explorar o potencial do Python explorando (e tendo muita dificuldade em definir o que usar) dicionários, listas de tuplas, sets… Além disso, melhorei bastante na manipulação de fluxos.

O desenvolvimento foi guiado pelo Gemini, claro, em vez de solicitar o código pronto, utilizei o Gemini como um gerente, se posso assim chamar, ele basicamente me guiou em um passo a passo do que fazer, ditando conceitos, apontava possíveis problemas (principalmente lógicos, foi o que mais teve) e separou todo o desenvolvimento em pequenos marcos/atividades, enquanto eu fiquei responsável realmente pela implementação e escrita do código. Acho que consegui tirar bastante proveito dessa forma de aprendizagem.

Honestamente, a parte mais difícil e frustrante foi a UI. Lidar com o gerenciamento de janelas, atualização de variáveis em tempo real e o posicionamento dos elementos no Tkinter foi um desafio imenso, além disso não consegui implementar a droga da scrollbar, que saco mexer com isso no Python, talvez seja mais fácil aprender do zero outra linguagem do que aprender a mexer no Tkinter.

Enfim, se alguém ver esse código (muito difícil) ou se o Igor futurista estiver lendo esse Readme de 10 anos atrás (mais provável), dê sugestões de melhorias ksks.
